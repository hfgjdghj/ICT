Parity check

Parity check is used for validation of data after transmission from one media to another,for example,transmitting data from hard disk to RAM or from one computer to another through a computer network.
There are two types of parity check: Odd parity check and even parity check

An even parity bit is set to 1 if the number of ones in the binary data is odd so that the total number of ones is even). An odd parity bit is set to 1 if the number of ones in the binary data is even so that the total number of ones is odd.If an odd number of bits in data are changed during transmission, the parity bit will indicate an error in transition has occured. The whole data block must be discarded and re-transmitted. Parity checking is a simple and effective means to validate data in data transmission.

Data Verification

When data is transcribed from one medium to another, there is always a risk of introducing errors. Data verification is a control used to check whether the inputted data matches that in the source document.

Input Data Twice

An operator inputs the data twice and allows the computer system to check the second entry against the first one. It reports any discrepencies and the operator is required to correct the answer manually.
When you are going to change the password of your e-mail account, you have to re-enter the password for confirmation.

Double Data Entry

Two operators enter the same source document into two different files. The computer compares the two files and reports any discrepencies between them. The discrepencies must be checked and corrected manually.

    File Access Modes

    Sequential Access Mode

    In a magnetic tape, data records are stored one by one along the lengthy magnetic strip. The magnetic tape drive has to wind the tape forward or backward until the desired record is just underneath the read/write head before the reading or writing operation can proceed. The seek time is long and unpredictable when records are retrieved randomly. The magnetic tape is not a common medium for storing frequently used data records. It is mainly used for backing up and recovering of database and data files in computer servers.

    Direct Access Mode

    The hard disk is the most common storage medium for secondary storage due to ts high reliability, speed and capacity. When data is read from or written to a hard disk, the magnetic disk plate (know as platter) rotates and the read/write head moves across the platter surface at the same time to the location containing the data. It is often used as secondary storage for operating systems, application softwares and data files.

    Adventage

    Once the target location is moved underneath the read/write head, the reading or writing process operation can proceed. This way of accessing data is called direct access. The seek time is much shorter and predictable, ranging from 8ms to 20ms. The hard disk is the most essential secondary storage device for a database system and database tables as well as all other computer software.
    Although floppy disks and optical disks like DVD-RAM and CD-RW also operate in direct access mode, they are limited in capacity and speed of data access. They are rarely used as storage media for database systems and frequently used data files.

    Number Representation in Computer Systems

    The binary number system is the only number system that a computer uses in storing numbers and performing arithmetic.

    Bit

    A bit is a single binary digit. It is the basic unit for storing data in a computer. With a single bit, any one of two distinct values can be represented, for example, 0 or 1, true or false, and on or off. If more distinct values need to be represented, a series of bits can be used. For example, an 8-bit binary number can represent 2n distict values.

    Byte

    A byte is a data unit used in computers. A byte consists of 8 bits and is the smallest addressable unit in a microprocessor.

    Word

    Modern computers can handle data stored in two or more bytes at a time. The natural unit of data used by a computer is called a word. The word length, or also known as word size, is measured in bits. Modern computers usually have a word length of 16, 32 or 64 bits. Word size refers to the number of bits processed by a computer's CPU in one go.

    Representation of Unsigned Binary Integer

    In a computer, when a word is used to store an unsigned binary integer, all bits in a word are used to represent the magnitude.

    Character Coding Systems

    Other than numeric data, a computer needs to store and process characters. The character coding system is a way to represent characters in a form that can be manipulated efficiently in a computer.

    ASCII

    American Standard Code for Information Interchange (ASCII) and Extended Binary-Coded Decimal Interchange Code (EBCDIC) are two common character coding systems for computers and some communication devices.Each code represents either a printable character or a non-printable character.
    Each ASCII character consists of 7 bit and it can contain 27 different characters.

    Chinese Character System

    The standard ASCII contains only 128 characters and the extended ASCII can have 256 characters. They are sufficient for english and Western European languages but are not large enough for Asian languages.There are more than 100000 Chinese characters and we use more than 10000 of them in our daily lives. We need some other character coding systems. Chinese characters are usually represented in Big5 code, Guo Biao (GB) code and unicode systems.
    Big5 code is mainly used to represent traditional Chinese and is very popular in the Chinese computer systems In Hong Kong, Macau and Taiwan. GB code is the standard code for simplified chinese. Both character coding systems use two bytes to represent one Chinese character. Since these two characters coding systems use different codes to represent the same Chinese characters, if a Chinese text file is processed with the wrong coding system, strange and meaningless characters will appear.

    Unicode

    Recently, Unicode has been used extensively to represent all the existing languages in the world. Unicode consists of a repertoire of about 100000 characters. It is an international standard code that sets the codes for commonly used characters in the world, including both traditional and simplified Chinese characters. The representations of Unicode known as code points are variable in length. They are made up of one to four bytes. The code points are carefully defined so that the most frequently used characters, such as the ASCII codes, are put at the front of the code table. As a result, these characters can be represented by a one-byte code point.
