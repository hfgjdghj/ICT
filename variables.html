Variables

First, Some common variables are integer, real, string, char, boolean.
Integer is a integral number ranging from -32768 to 32767.
Real can be decimals which will show in a format of 1.20000000000000E+001.
String is bunch of text,it can be numbers, everything on the ASCII table.
Char works like a string but can only store one character.
Boolean can be only true or false.

if you enter 'abcdefg', string can store 'abcdefg' , while char can only store 'a'.
if you enter 12345, Integer can store 12345 , while Real will store as 1.23450000000000E+004.
If you assign true to Boolean, it will output as TRUE.
Integer can store both positive and negative numbers, but not decimals.
Both Integer and Real can compare with decimals.
When you genarate a random number, use random(?) + ? and assign it to Integer.
Constants cannot be assign another value.
In ASCII table, A to Z is 65 to 90 and a to z is 97 to 122.

Common operators and Statements

Not and ( ) have the highest priority.
Then they are *, /, div, mod, and.
And then the +, -, or.
and then =, <=, >=, <, >.

Sqr(x) returns the square of x.
Sqrt(x) return the positive square root of x.
Abs(x) returns the absolute value of x.
Int(x) returns the integer part of x.(+/-)
Frac(x) returns the fractional part of x.
pi return value of pi and no parimeter is needed.
Odd(x) test if x is an odd number.
Pred(x) returns the predecessor of x(ordinal)
Succ(x) returns the successor of x.
Chr(x) returns the corresponding character of ASCII value of x.
Ord(x) returns the ordinal value of x.
Round(x) round real type x to integer type value.
Truncate(x) truncate a real type x to integer type value.
Case x of can provide various output for different value of x.
Gotoxy(x,y) moves the cursor to the corresponding coordinates.
Length(x) returns the length of string or array x.
Copy(string,start,length) copies part of the string
LeftStr(string,length) return characters from the left of the string.
RightStr(string,length) return characters from the right of the string.
Lowercase(string) return lowercase version of a string.
UpperCase(string) return uppercase of a string.
MidStr(string,start,length) copies a number of characters at a given position.
